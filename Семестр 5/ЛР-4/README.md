# Работа программы
## Возможности:
- GET /terms
Получение списка всех терминов
- GET /terms/{term}
Получение термина по названию
- POST /terms
Добавление нового термина
- PUT /terms/{term}
Обновление определения термина
- DELETE /terms/{term}
Удаление термина
